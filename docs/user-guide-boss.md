# Kachu 使用手冊 — 給老闆的操作指南

> 版本：beta v1  
> 更新日期：2026-04-28  
> 適用對象：第一次使用 Kachu 的企業用戶（老闆）

---

## Kachu 是什麼？

Kachu 是你的 AI 數位幕僚，住在你的 LINE 裡。

你只需要在 LINE 裡說話，Kachu 就會幫你：
- 把你傳的照片寫成 IG / Facebook 貼文草稿
- 幫你發 Google 商家動態
- 記住你的店家資訊（菜單、地址、特色）
- 回答顧客問題
- 整理評論並幫你擬回覆

**你不需要學任何 App，也不需要登入任何後台，就用 LINE 操作即可。**

---

## 第一步：加 Kachu 為 LINE 好友

1. 用手機掃描 QR code（由 Kachu 服務人員提供）
2. 或搜尋 LINE ID：`@kachu`（由服務人員告知正確 ID）
3. 加好友後，Kachu 會自動開始和你打招呼

---

## 第二步：完成初始設定（約 5 分鐘）

第一次和 Kachu 說話，它會引導你完成設定。**只需要照著回答，不需要特別記格式。**

整個流程如下：

### 步驟 1 — Kachu 問你店名
```
Kachu：請問你的店名是什麼？
你：鴻笙堂
```

### 步驟 2 — Kachu 問你行業
```
Kachu：謝謝！你的行業類型是什麼？
你：中醫調理
```

### 步驟 3 — Kachu 問你地址
```
Kachu：請告訴我你的地址或營業地點？
你：新北市泰山區仁義路222號
```
> 純網路店可以輸入「網路」

### 步驟 4 — 上傳現有資料（可跳過）

Kachu 會請你傳任何已有的資料，讓它更了解你的生意：
- 菜單或價目表的照片
- 舊的宣傳文字
- 任何你覺得有用的圖片或文件

**傳完後輸入「完成」，或直接輸入「跳過」進入下一步。**

### 步驟 5 — 三個問題

Kachu 會問你三個簡單的問題：
1. 你跟別家最不一樣的地方是什麼？
2. 你現在最大的困擾是什麼？
3. 今年你最想做的一件事是什麼？

照實回答即可，字數多少沒有限制。

### 設定完成！

設定完成後，Kachu 會說：
> 🎉 太好了！我已經了解你的生意了。  
> 試試看，傳一張你想發布的照片給我 📸

## 日常使用：你可以對 Kachu 說什麼

### 📸 傳照片 → 自動生成貼文

傳一張照片給 Kachu，它會自動分析照片並幫你寫貼文草稿。

**你怎麼做：**
1. 在 LINE 傳一張照片
2. Kachu 分析後，發給你一則草稿卡片
3. 卡片上有三個按鈕：**✅ 確認發布** / **✏️ 修改** / **❌ 不發了**

> 你也可以在傳照片時附上說明，例如：  
> 「這是今天的新品，幫我寫 IG 文，主題是母親節限定」

---

### ✍️ 叫 Kachu 寫文章

直接說你想要的內容：

```
幫我寫一篇 Google 動態，介紹我們的週年慶優惠
寫個 IG 貼文，主題是端午節
發一篇商家動態說我們本週五休息
```

Kachu 會幫你生成草稿，你再確認或修改。

---

### 📝 更新店家資訊

當店家資訊有變動，直接告訴 Kachu：

```
我們的營業時間改成早上 10 點到晚上 8 點
現在新增了外送服務
停車場已經不收費了
菜單新增一道鮮蝦粥，280 元
```

Kachu 會自動記住，之後回答顧客問題時會用到。

---

### ⭐ 處理 Google 評論

```
幫我看一下最新的評論
有新的負評，幫我擬回覆
```

> 注意：此功能需要先完成 Google Business 連結，由服務人員協助設定。

---

### 📊 查看業績報告

```
這週的業績怎麼樣？
上個月的訪客數
幫我看一下流量報告
```

> 注意：此功能需要先完成 Google Analytics 連結，由服務人員協助設定。

## 審批流程：如何確認或修改 Kachu 的草稿

當 Kachu 生成草稿後，會在 LINE 傳給你一張**卡片訊息**，上面有草稿內容和三個按鈕：

| 按鈕 | 代表意思 |
|---|---|
| ✅ 確認發布 | 這個草稿很好，直接發布 |
| ✏️ 修改 | 我想改改看，Kachu 請給我修改版 |
| ❌ 不發了 | 取消這次，不要發布 |

**直接點按鈕就好，不需要打字。**

## 常見問題

### Q：我說話 Kachu 沒有反應？
- 確認 Kachu bot 仍在你的好友清單中
- 確認 Kachu 服務正常（可詢問服務人員）

### Q：Kachu 寫的文章不符合我的風格？
傳照片或發指令時，可以加上更多說明：
```
幫我寫 IG 文，口吻要輕鬆一點，像在跟朋友說話
幫我寫，不要用太多 emoji
```
Kachu 會學習你的偏好，用得越多它越了解你。

### Q：我剛剛說的資訊說錯了，可以改嗎？
可以，直接說：
```
剛剛說的地址改一下，應該是221不是222
```

### Q：我不想要 Kachu 主動傳東西給我？
目前 Kachu 的主動通知包含每週營運建議。如有需要暫停，請告知服務人員調整設定。
